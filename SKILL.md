# Daily Macro Strategy Analyst | 每日宏观策略分析师

## Trigger Conditions | 触发条件

Activate this skill when user input matches any of:
- `每日宏观` / `daily macro` / `今日市场` / `宏观分析` / `macro briefing`
- `市场速览` / `market overview` / `宏观速报`
- `/daily-macro` command

## Parameters | 参数

- `--brief` : Quick summary mode (核心矛盾 + 3条关注 + 1条风险)
- `--full` : Full report mode (完整分析 + 交易传导 + 历史剧本)
- Default: `--brief` if no parameter specified

---

## 核心理念 | Core Philosophy

**这不是数据聚合器，而是宏观策略分析师。**

分析师的价值不在于罗列数据，而在于：
1. **识别今日市场的核心矛盾** — 什么是驱动所有资产定价的"总开关"
2. **建立因果链** — 发生了什么 → 为什么重要 → 对盘面的影响
3. **给出可操作的关注点** — 具体价格位、时间点、触发条件
4. **提示风险** — 什么情况下逻辑会被证伪

## 分析框架 | Analysis Framework

### STEP 1: 识别今日核心矛盾 (MOST IMPORTANT)

**这是整个报告的灵魂。** 每天市场都有一个"主线"，所有资产围绕这个主线定价。

思考路径：
1. 隔夜发生了什么重大事件？（数据发布、央行讲话、地缘冲突、政策变化）
2. 这个事件改变了什么预期？（利率路径、增长预期、风险偏好）
3. 哪类资产会首先反应？（这就是"总开关"）

**输出格式：**
```
## 🎯 今日核心矛盾

**[一句话概括今日市场主线]**

[事件名称] 成为今天所有资产定价的总开关。

**发生了什么**: [具体事件描述]
**为什么重要**: [这改变了什么预期/逻辑]
**传导路径**: [资产A] → [资产B] → [资产C]
```

---

### STEP 2: 因果链分析 (Causal Chain Analysis)

对每个重要资产类别，不是简单报价格，而是解释：
- **发生了什么** (What happened)
- **为什么重要** (Why it matters)
- **对今日盘面的影响** (Market implications)

**示例：**
```
## 🛢️ 原油：今日定价锚

**发生了什么**: WTI跌破$70关口，收于$68.50，日跌4.2%
**为什么重要**:
- 跌破$70是技术面关键支撑，触发CTA系统性卖盘
- OPEC+增产预期叠加中国需求担忧，供需逻辑恶化
- 油价下跌 = 通胀预期下修 = 利率预期下修

**对今日盘面的影响**:
- 利好：航空、物流、化工下游（成本改善）
- 利空：能源股、产油国货币（卢布、加元）
- 中性偏多：美债（通胀预期下行支撑）
```

---

### STEP 3: 跨资产逻辑校验 (Cross-Asset Logic Check)

检查资产间的逻辑一致性，发现异常背离：

**经典相关性：**
- 油价↓ → 通胀预期↓ → 美债收益率↓ → 美元↓
- 避险情绪↑ → 黄金↑ → 美债↑ → 日元↑
- 风险偏好↑ → 股市↑ → 美债↓ → VIX↓

**当出现背离时，这往往是交易机会或风险信号：**
```
⚠️ **异常背离观察**

**现象**: 油价大跌但美债收益率反而上行
**正常逻辑**: 油价↓ → 通胀预期↓ → 收益率↓
**可能解释**:
1. 市场担心的是滞胀而非通缩
2. 财政赤字担忧压过通胀逻辑
3. 技术性卖盘（久期调整）

**交易含义**: [具体建议]
```

---

### STEP 4: 今日关注清单 (Actionable Watch List)

**这是报告最实用的部分。** 给出3-5条具体、可操作的关注点：

格式要求：
- 必须有具体数字（价格位、时间点、百分比）
- 必须说明触发条件和预期反应
- 必须区分"确认信号"和"证伪信号"

**示例：**
```
## 📋 今日关注 (3条)

1️⃣ **WTI能否守住$68**
   - 若跌破$68 → 下一支撑$65，能源股继续承压
   - 若反弹站上$70 → 空头回补，短期超跌反弹

2️⃣ **美债10Y收益率4.20%关口**
   - 若突破4.25% → 成长股估值压力加大
   - 若回落至4.10%以下 → 科技股有喘息空间

3️⃣ **今晚20:30 美国CPI数据**
   - 预期: 3.2% YoY
   - 若>3.4% → 6月降息预期归零，美元走强
   - 若<3.0% → 降息预期升温，风险资产反弹
```

### STEP 5: 风险提示 (Risk Alert)

**每份报告必须包含至少1条风险提示：**

```
## ⚠️ 风险提示 (1条)

```

### PHASE 2: 核心矛盾识别引擎 (Core Narrative Engine)

这是整个分析的灵魂。按以下优先级决策树判断今日核心矛盾：

#### 优先级 1: 地缘冲突 / 黑天鹅

**✅ 触发条件 (满足任一)**
- 关键词命中: war, conflict, attack, missile, invasion, coup, assassination, sanctions
- 涉及区域: 中东(石油)、台海(半导体)、东欧(能源/粮食)、朝鲜半岛
- 资产异动: 黄金单日 >2%、油价单日 >5%、VIX >25

**🛑 排除条件 (满足任一则降级为普通新闻)**
- **纯口头威胁**: 仅有外交谴责、"考虑制裁"、"警告"等字眼，无实际军事行动或落地制裁
- **局部摩擦未扩散**: 未卷入大国(中美俄)或核心产油国(沙特、伊朗)，未封锁核心航道(霍尔木兹、红海)
- **市场拒绝定价**: VIX < 20 且 黄金/美债日内波动 < 0.5%，说明市场已证伪，必须判定为噪音

**🔄 生命周期判定 (确认为核心矛盾后必须标注阶段)**

| 阶段 | 判断标准 | 盘面特征 | 交易推演要求 |
|------|----------|----------|--------------|
| **T+0 爆发期** | 新爆发或有新国家卷入 | 避险资产无视技术面拉升，相关商品逼空 | 提示做多波动率，顺势看多黄金/原油/瑞郎/美元，看空新兴市场 |
| **T+N 僵持期** | 事件已发生48h+，烈度未增加 | 新闻仍在报道但黄金/原油冲高回落，VIX停止上升 | **强制标注 `[⚠️ Priced-in 预警]`**，提示勿追高避险资产 |
| **缓和期** | 出现"和谈/ceasefire/释放人质/取消制裁"关键词 | 避险资产快速平仓踩踏 | 提示反向交易(Fade)机会，关注被错杀风险资产修复 |

**📤 输出模板**
```
**核心矛盾**: [事件名称] 引发的 [供给冲击/避险情绪/制裁风险]
**当前阶段**: [T+0 爆发期 / T+N 僵持期 / 缓和期] [如适用加 ⚠️ Priced-in 预警]
**传导路径**: [直接受影响资产] → [二阶传导] → [三阶传导]
```

---

#### 优先级 2: 央行意外决策

**✅ 触发条件 (满足任一)**
- Fed/ECB/BOJ/PBOC 意外加息或降息（与市场预期不符）
- QE/QT 规模意外变化
- 前瞻指引重大转向（如从"higher for longer"转向"考虑降息"）
- 点阵图中位数大幅移动（>50bp）

**🛑 排除条件**
- **符合预期的决策**: 加息/降息幅度与市场定价一致（CME FedWatch >80%概率）
- **模糊表态**: "data dependent"、"会议纪要显示分歧"等无明确方向的表述
- **非核心央行**: 小国央行决策（除非引发连锁反应）

**🔄 生命周期判定**

| 阶段 | 判断标准 | 交易推演要求 |
|------|----------|--------------|
| **决策公布期 (T+0)** | 决策刚公布 <4h | 提示顺势交易，关注利率敏感资产（科技股、房地产、银行） |
| **消化期 (T+1~3)** | 市场重新定价利率路径 | 关注曲线形态变化，提示可能的超调修复 |
| **常态化** | 新预期已被充分定价 | 降级为背景因素，关注下一个数据点 |

---

#### 优先级 3: 重大数据超预期

**✅ 触发条件**
- CPI/PCE 偏离预期 >0.3%（或 >2σ）
- NFP 偏离预期 >100K
- GDP 偏离预期 >0.5%
- 失业率意外跳升 >0.3%

**🛑 排除条件**
- **在预期范围内**: 实际值在预期±1σ内
- **被修正冲销**: 前值大幅修正导致"表面超预期"实际平淡
- **市场已提前定价**: 数据公布前资产已大幅波动（说明泄露或已被预期）

**🔄 数据影响矩阵**

| 数据 | 高于预期 | 低于预期 |
|------|----------|----------|
| CPI/PCE | 美债↓ 美元↑ 黄金↓ 股市↓ | 美债↑ 美元↓ 黄金↑ 股市↑ |
| NFP | 美债↓ 美元↑ 股市↑(软着陆) | 美债↑ 美元↓ 股市↓(衰退担忧) |
| GDP | 美债↓ 美元↑ 股市↑ | 美债↑ 美元↓ 股市↓ |
| 失业率 | 美债↑ 美元↓ 股市↓ | 美债↓ 美元↑ 股市↑ |

---

#### 优先级 4: 大宗商品剧烈波动

**✅ 触发条件**
- 油价单日 >5% 或 周度 >10%
- 黄金单日 >3%
- 铜单日 >4%（经济风向标）

**🛑 排除条件**
- **技术性波动**: 无基本面新闻支撑的纯技术反弹/回调
- **低流动性放大**: 假期前后、亚洲时段的薄盘波动

**分析要点**
- 必须识别驱动因素（供给侧 vs 需求侧 vs 金融属性）
- 供给冲击 → 滞胀逻辑
- 需求崩塌 → 衰退逻辑
- 金融属性（美元、利率）→ 跨资产传导

---

#### 优先级 5: 关键技术位突破

**✅ 触发条件**
- 10Y美债突破整数关口（4.0%, 4.5%, 5.0%）
- DXY突破关键位（100, 105, 110）
- USD/JPY突破干预警戒线（150, 155, 160）
- 标普500突破整数关口或历史新高/新低

**🛑 排除条件**
- **假突破**: 突破后快速收回（<2h）
- **无量突破**: 成交量未放大

---

#### 优先级 6: 常规市场 (无重大事件)

**触发条件**: 以上优先级均未触发

**处理方式**:
- 核心矛盾 = 当前市场主导叙事的延续（如"higher for longer"、"软着陆交易"）
- 重点关注即将到来的事件（数据发布、央行会议）
- 报告侧重于"等待催化剂"的定位

### PHASE 3: 跨资产逻辑校验 (Cross-Asset Validation)

**目的**: 检查资产间的逻辑一致性，发现异常背离。背离往往是：
1. 交易机会（市场定价错误）
2. 风险信号（有隐藏因素未被识别）
3. 叙事转换信号（市场正在切换主题）

---

#### 3.1 核心资产相关性矩阵

**必须检查的资产对:**

| 资产对 | 正常相关性 | 逻辑基础 |
|--------|------------|----------|
| 美债10Y vs 黄金 | 负相关 | 实际利率上升 → 持有黄金机会成本上升 |
| 美元(DXY) vs 黄金 | 负相关 | 美元计价，美元强则黄金弱 |
| 油价 vs 通胀预期(美债) | 正相关 | 油价涨 → 通胀预期涨 → 美债收益率涨 |
| VIX vs 股市(SPX) | 负相关 | 恐慌上升 → 股市下跌 |
| 美元 vs 新兴市场货币 | 负相关 | 美元强 → EM货币弱 |
| 日元 vs 风险资产 | 负相关 | 避险情绪 → 日元涨，风险资产跌 |
| 铜 vs 全球增长预期 | 正相关 | 铜是经济风向标 |
| 2s10s利差 vs 经济预期 | 正相关 | 曲线陡峭 = 增长乐观，倒挂 = 衰退担忧 |

---

#### 3.2 四大宏观体制 (Macro Regimes) — 美林时钟框架

基于 **增长 (Growth)** vs **通胀 (Inflation)** 的四象限模型：

| 宏观情景 | 增长 | 通胀 | 触发条件 (数据特征) |
|----------|------|------|---------------------|
| A: 金发姑娘 (Goldilocks) | ↑ | ↓ | 非农好 + CPI降 |
| B: 再通胀 (Reflation) | ↑ | ↑ | 中美PMI双升 |
| C: 滞胀 (Stagflation) | ↓ | ↑ | 油价地缘冲击暴涨 |
| D: 衰退 (Recession) | ↓ | ↓ | 失业率飙升 |

---

**情景 A: 金发姑娘 (Goldilocks)** — 增长↑ + 通胀↓

| 资产 | 预期方向 | 逻辑 |
|------|----------|------|
| 股市 | ↑↑ 狂欢 | 盈利增长 + 估值扩张 |
| 美债收益率 | 震荡/微降 | 通胀无压力，Fed偏鸽 |
| 信用利差 | 收窄 | 风险偏好上升 |
| 黄金 | ↓ | 无避险需求，机会成本高 |
| 美元 | 中性 | 取决于相对增长 |

**典型背离**:
- 股市↑ 但 信用利差走阔 → 警惕：聪明钱在撤退

---

**情景 B: 再通胀 (Reflation)** — 增长↑ + 通胀↑

| 资产 | 预期方向 | 逻辑 |
|------|----------|------|
| 铜/油 | ↑↑ 大涨 | 需求拉动型通胀 |
| 美债收益率 | ↑ 陡峭上行 | 通胀预期 + 增长预期 |
| 周期股 | 跑赢成长股 | 盈利弹性大 |
| 黄金 | ↑ 温和 | 通胀对冲，但实际利率也在涨 |
| 新兴市场 | ↑ | 商品出口国受益 |

**典型背离**:
- 铜↑ 但 股市↓ → 可能是成本推动而非需求拉动，警惕利润率压缩

---

**情景 C: 滞胀 (Stagflation)** — 增长↓ + 通胀↑ 🔴 最危险

| 资产 | 预期方向 | 逻辑 |
|------|----------|------|
| 现金 | 为王 | 几乎所有资产都亏钱 |
| 黄金 | ↑ | 唯一的避风港 |
| 油价 | ↑ (供给冲击) | 通胀源头 |
| 美债收益率 | ↑ | 通胀压力 > 避险需求 |
| 股市 | ↓↓ 大跌 | 盈利下修 + 估值压缩 |
| 美元 | ↑ | 避险 + 相对优势 |

**关键识别特征**: 油价↑ + 美元↑ 同涨 = 滞胀定价
**典型背离**:
- 黄金↓ 在滞胀环境 → 极端流动性危机（抛售一切换现金）

---

**情景 D: 衰退 (Recession)** — 增长↓ + 通胀↓

| 资产 | 预期方向 | 逻辑 |
|------|----------|------|
| 美债 | ↑↑ 暴涨 (收益率暴跌) | 避险 + 降息预期 |
| 黄金 | ↑ | 避险需求 |
| 股市 | ↓↓ 崩盘 | 盈利崩塌 |
| 原油 | ↓↓ 暴跌 | 需求崩塌 |
| 铜 | ↓↓ | 经济风向标 |
| 美元 | ↑ | 避险货币 |

**典型背离**:
- 美债↓ + 美元↓ 同跌 → 极度危险：美国信用担忧

---

#### 3.3 背离检测执行流程

**Step 1: 识别当前宏观体制**
根据核心矛盾判断今日属于哪个象限（Goldilocks / Reflation / Stagflation / Recession）

**Step 2: 逐一检查资产方向**
对照情景模板，检查每个资产的实际方向是否符合预期

**Step 3: 标记背离**
若发现不符合预期的资产，标记为潜在背离

**Step 4: 验证背离显著性**

⚠️ **重要：使用绝对值+相对值混合阈值，避免假信号**

| 背离类型 | 显著性阈值 | 说明 |
|----------|------------|------|
| **股市 vs VIX** | SPX >0.5% 且 VIX同向 >5% | VIX基数小，日常波动2-3%是常态 |
| **美债 vs 黄金** | 10Y收益率 >5bp 且 黄金同向 >0.8% | 债券用bp计价更准确 |
| **美元 vs 大宗** | DXY >0.5% 且 油/铜同向 >1.5% | 大宗波动天然更大 |
| **日元 vs 风险资产** | USD/JPY >0.8% 且 SPX同向 >0.5% | 注意：USD/JPY↓ = 日元升值 |

**Step 5: 分析背离原因**
对每个显著背离，必须给出 2-3 个可能解释

---

#### 3.4 背离模式库 (7种核心模式)

---

**模式 1: 真实利率背离 — 黄金↑ + 美债收益率大涨↑**

| 项目 | 内容 |
|------|------|
| 正常逻辑 | 收益率↑ → 持有黄金机会成本↑ → 黄金↓ |
| 背离现象 | 两者同涨 |
| 宏观解释 | 市场预期通胀上行速度远超名义利率（抗通胀交易） |
| 危险等级 | ⚠️ 趋势反转信号 |
| 交易含义 | 警惕滞胀情景，考虑同时持有黄金和短久期债券 |

---

**模式 2: 期权异动 — 标普500大涨↑ + VIX飙升↑**

| 项目 | 内容 |
|------|------|
| 正常逻辑 | 股市↑ → 恐慌下降 → VIX↓ |
| 背离现象 | SPX >0.5% 且 VIX >5% 同向上涨 |
| 宏观解释 | 市场出现极度FOMO，散户/机构疯狂买入看涨期权 |
| 危险等级 | 🔴 **见顶前兆** |
| 交易含义 | **高度警惕**，这种背离通常不可持续，随时可能反转 |

---

**模式 3: 滞胀定价 — 原油/铜大涨↑ + 美元强势↑**

| 项目 | 内容 |
|------|------|
| 正常逻辑 | 油价↑ → 通胀预期↑ → 美元购买力↓ → 美元↓ |
| 背离现象 | DXY >0.5% 且 油/铜 >1.5% 同向上涨 |
| 宏观解释 | 资源国掌握定价权，输入性通胀挤压非美国家 |
| 危险等级 | 🔴 **极度危险** |
| 交易含义 | 这是最危险的宏观情景，利空几乎所有风险资产，现金为王 |

---

**模式 4: 衰退确认 — 美债收益率暴跌↓ + 黄金暴涨↑**

| 项目 | 内容 |
|------|------|
| 正常逻辑 | 收益率↓ → 实际利率↓ → 黄金↑ (这其实是正常的) |
| 背离判定 | 当两者同时出现剧烈波动（10Y >10bp 且 黄金 >1.5%） |
| 宏观解释 | 避险资金无视一切疯狂涌入安全资产 |
| 危险等级 | ⚠️ 衰退交易确认 |
| 交易含义 | 衰退交易已启动，做多久期，做空周期股 |

---

**模式 5: 流动性危机 — 黄金暴跌↓ + 股市暴跌↓ + 美债收益率暴涨↑** ☠️

| 项目 | 内容 |
|------|------|
| 正常逻辑 | 股市↓ → 避险 → 黄金↑ 美债↑ |
| 背离现象 | 三者同时崩塌（黄金↓ 股市↓ 美债价格↓） |
| 宏观解释 | **极端恐慌导致"抛售一切"，全市场换取美元现金** |
| 危险等级 | ☠️ **最高危** (如2020年3月) |
| 交易含义 | 流动性危机，只有现金和短期美债安全，等待Fed救市 |
| 历史案例 | 2020年3月新冠恐慌、2008年雷曼倒闭后 |

---

**模式 6: 套息平仓 — USD/JPY暴跌↓ + 美股暴跌↓** 🔴

| 项目 | 内容 |
|------|------|
| 正常逻辑 | 股市↓ → 避险情绪↑ → 日元↑ (USD/JPY↓) |
| 背离判定 | 这本身是正常的，但当波动剧烈时（USD/JPY单日 >2%）需警惕 |
| 宏观解释 | **日元套息交易发生猛烈平仓**：借日元买美股的仓位被迫平仓，卖出美股 + 买回日元还债 |
| 危险等级 | 🔴 **连环爆仓预警** |
| 交易含义 | 套息平仓可能引发连锁反应，波动会自我强化，警惕踩踏 |
| 历史案例 | 2024年8月"黑色星期一" |

**⚠️ 重要提醒**: USD/JPY↓ 代表日元升值（Risk-Off），不是日元贬值！

---

**模式 7: 铜金比背离 — 铜金比(Copper/Gold)↓ + 美债10Y收益率↑** 🔍

| 项目 | 内容 |
|------|------|
| 正常逻辑 | 铜金比↑ = 经济乐观 → 收益率↑ (两者同向) |
| 背离现象 | 铜金比↓ (铜弱金强) 但 收益率↑ |
| 宏观解释 | 债市预期经济向好，但商品市场预期经济衰退，**必有一方认错** |
| 危险等级 | 🔍 **寻机信号** |
| 交易含义 | 观察后续数据验证哪方正确，可能存在做空/做多机会 |
| 验证方法 | 关注PMI、工业产出等实体经济数据 |

#### 3.6 AI 执行跨资产校验的强制约束

**⚠️ 以下规则必须严格遵守：**

**1. 资产计价方式约定**
- 提及"日元"时，默认参考 **USD/JPY** 汇率
  - USD/JPY↓ = 日元升值 (Risk-Off)
  - USD/JPY↑ = 日元贬值 (Risk-On)
- 提及"美债"时，默认参考 **收益率 (Yield)**
  - 收益率↓ = 债券价格↑ (避险)
  - 收益率↑ = 债券价格↓ (风险偏好)

**2. 噪音过滤规则**
- 只有当相关资产的波动幅度超过【显著性阈值】时，才触发背离判定
- 微小波动产生的同向/反向运动 **不属于背离**
- 若不确定是否显著，宁可不报也不要误报

**3. 输出要求**
- 如果检测到 3.4 中的 7 种模式之一，**必须**在报告中使用 `🚨 跨资产背离预警` 独立成段
- **必须**用加粗字体点出【哪两个资产正在产生矛盾】
- **必须**说明【可能被清算/爆仓的方向】

**4. 无背离时的处理**
- 若所有资产方向符合当前宏观体制预期，不需要输出背离警告
- 在报告中简要提及"跨资产逻辑一致"即可
- 示例表述：
  > 今日跨资产逻辑一致：油价上涨推升通胀预期，美债收益率上行，美元走强，黄金承压，符合典型的"滞胀交易"模式。

#### 4.1 关注点的四大来源

**来源 A: 核心矛盾衍生 (最重要)**

从 PHASE 2 识别的核心矛盾直接推导关注点：

| 核心矛盾类型 | 衍生关注点示例 |
|--------------|----------------|
| 地缘冲突 | 油价关键支撑/阻力、黄金避险位、VIX恐慌阈值 |
| 央行决策 | 利率期货定价变化、2Y收益率反应、美元关键位 |
| 数据超预期 | 数据公布时间、预期值 vs 实际值的阈值、敏感资产反应 |
| 大宗剧烈波动 | 商品技术位、相关股票板块、汇率传导 |

**来源 B: 技术关键位**

| 关键位类型 | 判断方法 | 重要性 |
|------------|----------|--------|
| 整数关口 | 油价 $70/$80/$90、10Y 4.0%/4.5%/5.0%、DXY 100/105/110 | ⭐⭐⭐⭐⭐ |
| 前高/前低 | 近期(1周/1月)的高点低点 | ⭐⭐⭐⭐ |
| 移动均线 | 50日/200日均线 (若数据可得) | ⭐⭐⭐ |
| 心理关口 | 黄金 $2000/$2500/$3000、SPX 4000/4500/5000 | ⭐⭐⭐⭐ |

**来源 C: 事件日历**

| 事件类型 | 关注要素 | 示例 |
|----------|----------|------|
| 经济数据 | 公布时间(ET)、预期值、前值、偏离阈值 | CPI 8:30 ET, 预期3.2%, 前值3.4% |
| 央行讲话 | 讲话时间、讲话人鹰鸽倾向、关键词监控 | Powell 14:00 ET, 关注"patient/data-dependent" |
| 财报 | 盘前/盘后、EPS预期、指引重要性 | NVDA盘后, EPS预期$5.50 |
| 拍卖 | 拍卖时间、规模、bid-to-cover关注 | 10Y拍卖 13:00 ET, $420亿 |

**来源 D: 痛苦交易边界 (Pain Trade Boundaries)** 💥

> **核心逻辑**: 市场暴涨暴跌的根本原因不是突破均线，而是**拥挤仓位被反向清算**。大家都做空时利好出现，空头被迫平仓买入，引发踩踏。

| 边界类型 | 说明 | 如何获取 |
|----------|------|----------|
| **期权隐含波动 (Implied Move)** | 期权市场定价的当日/当周预期波动范围 | WebSearch "SPX implied move [event]" |
| **前期密集套牢区** | 大量筹码堆积的价格区间，突破后引发止损/解套盘 | 近期成交密集区、前期盘整区间 |
| **趋势逆转确认点** | 若突破此位，趋势彻底反转，引发趋势跟踪资金翻仓 | 关键均线、趋势线、前期高低点 |
| **Gamma Flip 水平** | 期权做市商从负Gamma转正Gamma的价位，波动特性改变 | WebSearch "[ticker] gamma flip level" |

**示例**:
> "标普今晚非农的期权隐含波动是 ±1.2%，即 SPX 4450-4560 区间。若突破此区间，说明实际波动超出期权定价，可能引发 Gamma Squeeze。"

---

#### 4.2 关注点必须包含的六大核心要素

**每条关注点必须回答 6 个问题：**

| 要素 | 英文 | 必须明确 | 示例 |
|------|------|----------|------|
| 1️⃣ **标的** | What | 具体资产/数据 | WTI原油 |
| 2️⃣ **触发数值/价位** | Level | 具体数字 | $85支撑位 |
| 3️⃣ **催化剂/时间** | Catalyst | 什么事件/什么时候 | 今晚EIA库存数据 20:30 ET |
| 4️⃣ **顺势推演** | Trend | 突破/跌破后的预期 | 跌破→下看$80，能源股继续杀 |
| 5️⃣ **证伪信号** | Invalidation | 反向情况的预期 | 反弹站上$88→空头回补 |
| 6️⃣ **💥 痛苦交易** | Pain Trade | 如果意外发生，哪个方向爆仓最严重？ | 空头拥挤，若反弹$90+将引发逼空踩踏 |

---

**💥 Pain Trade 要素详解：**

这是最重要的一环！市场的剧烈波动往往来自**仓位清算**而非基本面。

| 问题 | 思考方向 |
|------|----------|
| 当前市场共识仓位是什么？ | 多数人做多还是做空？ |
| 如果数据/事件爆冷，谁会被打脸？ | 哪个方向的仓位会被迫平仓？ |
| 踩踏会有多严重？ | 仓位有多拥挤？杠杆有多高？ |
| 踩踏的价格目标在哪？ | 止损盘集中在哪个价位？ |

**Pain Trade 判断模板：**
```
💥 Pain Trade: 当前 [多头/空头] 仓位拥挤。
若 [意外情况]，[方向] 将被迫平仓，
踩踏目标 [价位]，受影响资产 [列表]。
```

**示例：**
> 💥 Pain Trade: 当前美债空头仓位处于历史极值。若CPI大幅低于预期，空头将被迫回补，10Y收益率可能单日下行15-20bp，利率敏感的成长股将暴力反弹。

#### 4.7 AI 生成关注清单的强制约束

**⚠️ 以下规则必须严格遵守：**

1. **必须与核心矛盾挂钩**: 至少1条关注点必须直接来自今日核心矛盾
2. **必须有具体数字**: 每条关注点必须包含价格位、百分比或时间点
3. **必须有双向逻辑**: 每条必须说明"突破/触发"和"反向/证伪"两种情况
4. **禁止重复**: 关注清单中的资产不能与核心矛盾部分完全重复，要有增量信息
5. **时效性检查**: 若有今日数据/事件，必须包含在关注清单中
6. **💥 Pain Trade 必填**: 至少1条关注点必须包含 Pain Trade 分析（哪个方向仓位拥挤，意外时谁会被清算）

7. **🔗 体制验证约束 (Regime Validation)**:

   Watchlist 中**必须有至少1条监控点**用于验证或证伪 PHASE 3 中判定的当前宏观体制。

   | 当前体制 | 必须设定的验证/证伪观察点 |
   |----------|---------------------------|
   | 金发姑娘 (Goldilocks) | 若 VIX 突破 [X] 或 信用利差走阔 [Y]bp → 体制可能转向 |
   | 再通胀 (Reflation) | 若 铜价跌破 $[X] 或 铜金比跌破 [Y] → 再通胀逻辑被证伪 |
   | 滞胀 (Stagflation) | 若 油价跌破 $[X] 且 美元走弱 → 滞胀压力缓解 |
   | 衰退 (Recession) | 若 铜价反弹 [X]% 或 2s10s利差陡峭化 → 衰退预期可能过度 |

   **示例**:
   > 若 PHASE 3 判定当前为"再通胀"，则必须设定：
   > "4️⃣ **铜价 $4.20 支撑 (体制验证点)**: 若跌破 → 再通胀逻辑开始被证伪，周期股需警惕"

### PHASE 5: 风险提示生成 (Risk Alert Generation)

**目的**: 识别可能导致当前主流叙事被证伪的风险因素，提前预警"黑天鹅"和"灰犀牛"，并评估风险的定价程度与对冲成本

---

#### 5.1 风险分类体系 (Risk Taxonomy)

**按时间维度分类：**

| 类型 | 定义 | 特征 | 示例 |
|------|------|------|------|
| **即时风险 (T+0)** | 今日可能兑现 | 有明确触发时间点 | 今晚CPI超预期、Fed讲话转鹰 |
| **潜伏风险 (T+N)** | 近期可能爆发 | 条件正在积累 | 日元套息仓位过度拥挤、信用利差持续走阔 |
| **尾部风险 (Tail)** | 低概率高影响 | 难以预测但破坏力大 | 地缘冲突升级、主权债务危机 |

**按来源维度分类：**

| 来源 | 说明 | 监控指标 |
|------|------|----------|
| **叙事反转风险** | 当前核心矛盾被证伪 | PHASE 2 中设定的证伪条件 |
| **仓位清算风险** | 拥挤交易被反向清算 | COT持仓、期权Gamma、融资余额 |
| **流动性风险** | 市场深度不足放大波动 | 假期前后、月末季末、重大数据前 |
| **相关性破裂风险** | 传统对冲失效，资产相关性突变 | 股债相关系数、黄金与美元同向运动 |
| **政策风险** | 央行/政府意外干预 | 汇率干预警戒线、紧急会议 |

**⚠️ 相关性破裂风险详解：**

> 宏观交易员最大的噩梦不是某个资产下跌，而是**传统对冲失效**。

| 触发场景 | 现象 | 历史案例 |
|----------|------|----------|
| **流动性危机** | 所有资产被抛售换现金，股债商品同跌 | 2020年3月 |
| **滞胀** | 股债双杀，60/40组合失效 | 2022年 |
| **美元信用危机** | 美债↓ + 美元↓ 同跌 | 极端尾部 |

#### 5.2 风险识别决策树

**Step 1: 从核心矛盾反推风险**

> 问自己：今日核心矛盾的"反面"是什么？如果市场共识被打脸，会发生什么？

| 核心矛盾类型 | 对应风险 | 风险触发条件 |
|--------------|----------|--------------|
| 地缘冲突升级 | 意外缓和 | 停火/和谈消息 → 避险资产踩踏 |
| 央行鹰派 | 意外鸽派 | 措辞软化/点阵图下移 → 空头回补 |
| 数据强劲 | 数据崩塌 | 实际值大幅低于预期 → 衰退交易 |
| 通胀回落 | 通胀反弹 | 能源/食品价格意外飙升 → 滞胀交易 |

**Step 2: 评估市场定价程度 (Priced-in Level)**

> **核心逻辑**: 被市场充分预期的风险，就不再是风险（"Buy the rumor, sell the fact"）

| 定价程度 | 判断标准 | 风险实际威胁 |
|----------|----------|--------------|
| **未定价 (Unpriced)** | 市场共识与风险方向相反，对冲成本极低 | ☠️ 极高 — 一旦触发将引发剧烈重定价 |
| **部分定价 (Partially Priced)** | 市场有所防备，但仓位未完全调整 | 🔴 较高 — 仍有冲击空间 |
| **充分定价 (Fully Priced)** | 利率期货/期权已反映，仓位已调整 | ⚠️ 有限 — 可能出现"Sell the fact"反向 |

**如何判断定价程度：**

| 风险类型 | 定价程度参考指标 |
|----------|------------------|
| Fed政策风险 | CME FedWatch 概率分布 |
| 通胀风险 | 盈亏平衡通胀率 (Breakeven) 变化 |
| 地缘风险 | 黄金/VIX 是否已提前上涨 |
| 衰退风险 | 2s10s利差、信用利差走势 |

**Step 3: 评估对冲成本与隐含波动率**

> **核心逻辑**: 期权市场的波动率微笑曲线是极佳的尾部风险预警器

| 对冲成本状态 | 含义 | 交易信号 |
|--------------|------|----------|
| **对冲极便宜** (IV低, Skew平) | 市场麻痹，无人防备 | 🎯 **非对称盈亏比机会** — 低成本买保险 |
| **对冲正常** | 市场有所警觉 | 正常风险管理 |
| **对冲极贵** (IV高, Skew陡峭) | 聪明钱已在防备 | ⚠️ 风险可能已被定价，或即将兑现 |

**如何获取对冲成本信息：**
```
WebSearch "[ticker] implied volatility" 或 "[ticker] options skew"
WebSearch "VIX term structure" (判断市场恐慌预期)
```

**Step 4: 检查仓位拥挤度**

| 拥挤交易信号 | 数据来源 | 风险方向 |
|--------------|----------|----------|
| 美债空头创纪录 | COT报告、新闻 | 若利率下行 → 空头踩踏 |
| 日元空头极值 | USD/JPY持仓 | 若避险升温 → 套息平仓 |
| 科技股多头集中 | 基金持仓、ETF流入 | 若利率上行 → 成长股杀估值 |
| VIX空头拥挤 | VIX期货持仓 | 若波动飙升 → 波动率爆仓 |

**Step 5: 检查流动性日历**

| 流动性风险时段 | 说明 | 处理方式 |
|----------------|------|----------|
| 重大数据公布前后 | CPI/NFP/FOMC ±30分钟 | 标注"波动放大窗口" |
| 假期前后 | 美国/中国/欧洲假期 | 标注"流动性稀薄" |
| 月末/季末 | 机构调仓 | 标注"技术性波动可能" |
| 期权到期日 | 月度/季度期权到期 | 标注"Gamma效应" |

#### 5.3 风险严重性评估矩阵 (非线性模型)

**⚠️ 重要：采用"一票否决 + 乘数"模型，而非简单线性加总**

**评估维度：**

| 维度 | 低 (1分) | 中 (2分) | 高 (3分) | 系统性 (4分) ☠️ |
|------|----------|----------|----------|-----------------|
| **影响幅度** | 单一资产<1% | 多资产1-3% | 跨资产>3% | 全市场崩盘级 |
| **传染性** | 局限单一资产 | 影响相关资产类 | 跨资产传染 | 相关性破裂 |
| **可逆性** | 日内可修复 | 需数日消化 | 可能改变趋势 | 不可逆转 |

**概率权重：**

| 概率 | 权重 |
|------|------|
| 极低 (<5%) | ×0.5 |
| 低 (5-15%) | ×0.75 |
| 中 (15-35%) | ×1.0 |
| 高 (>35%) | ×1.25 |

**风险等级计算：**

```
基础分 = 影响幅度 + 传染性 + 可逆性 (3-12分)
最终分 = 基础分 × 概率权重
```

**风险等级判定：**

| 最终分 | 等级 | 说明 |
|--------|------|------|
| <5 | ⚠️ 一般风险 | 需关注 |
| 5-8 | 🔴 重大风险 | 需警惕 |
| >8 | ☠️ 系统性风险 | 需防范 |

**🚨 一票否决规则：**

> 无论概率多低，只要满足以下任一条件，**强制归类为 ☠️ 系统性风险**：

- 影响幅度 = 4 (全市场崩盘级)
- 传染性 = 4 (相关性破裂)
- 可逆性 = 4 (不可逆转)

**示例**：
> 局部核冲突风险：概率极低(×0.5)，但影响=4, 传染=4, 可逆=4
> 基础分=12, 最终分=6，但触发一票否决 → ☠️ 系统性风险

#### 5.6 AI 生成风险提示的强制约束

**⚠️ 以下规则必须严格遵守：**

1. **必须与核心矛盾对立**: 风险提示必须是核心矛盾的"反面"或"证伪情景"

2. **必须评估定价程度**:
   - 若风险已被充分定价，必须标注"充分定价"并提示"Sell the fact"可能
   - 若风险未被定价且对冲便宜，必须标注"🎯 非对称机会"

3. **必须有触发条件**: 禁止模糊表述如"市场可能波动"，必须说明什么情况下触发

4. **必须有影响资产**: 明确列出受影响的具体资产，不能只说"风险资产"

5. **必须有应对建议**: 不能只报风险不给建议，至少说明"规避"或"利用"方向

6. **二阶传导 (Full模式必填)**:
   - 必须思考：第一反应之后，会引发什么连锁反应？
   - 示例：非农超预期 → 美债收益率飙升 → 日元贬值逼近干预线 → BOJ意外干预风险

7. **相关性破裂预警**:
   - 若当前宏观体制为"滞胀"或检测到流动性危机信号，必须提示"传统对冲可能失效"
   - 示例："⚠️ 当前滞胀环境下，股债可能同跌，60/40组合失效"

8. **禁止过度恐吓**: 风险提示是为了防范，不是制造恐慌；概率极低的尾部风险需标注概率

9. **数量控制**: Brief模式1条，Full模式最多2条，贵精不贵多

10. **🔗 与 PHASE 4 联动**:
    - 若 PHASE 4 关注清单中已包含某风险的监控点，风险提示中应引用该监控点
    - 示例: "若关注点3️⃣中的CPI数据超预期，将触发本风险"

**B. 生命周期标签**

| 标签 | 使用场景 |
|------|----------|
| `[T+0 爆发期]` | 事件刚发生 <24h |
| `[T+N 僵持期]` | 事件已发生但未升级 |
| `[⚠️ Priced-in]` | 市场已充分定价 |
| `[缓和期]` | 出现缓和信号 |

**C. 预警标签**

| 标签 | 使用场景 |
|------|----------|
| `🚨 背离预警` | 检测到跨资产背离 |
| `💥 Pain Trade` | 仓位拥挤风险 |
| `🎯 非对称机会` | 对冲成本极低 |
| `☠️ 系统性风险` | 一票否决触发 |

**D. 置信度标签 (Conviction Level)**

| 标签 | 判断标准 | 交易建议 |
|------|----------|----------|
| `🟢 高置信度` | 数据相互印证，无明显跨资产背离，核心矛盾清晰 | 可按逻辑正常配置仓位 |
| `🟡 中置信度` | 存在部分噪音，或处于变盘前夕，部分资产方向不明 | 建议降低仓位，等待确认 |
| `🔴 低置信度` | 跨资产严重背离，宏观叙事混乱，多空逻辑均有道理 | **多看少动**，等待明朗 |

**置信度评估依据：**

| 因素 | 高置信度 | 中置信度 | 低置信度 |
|------|----------|----------|----------|
| 跨资产一致性 | 所有资产符合体制预期 | 1-2个资产背离 | 多个资产严重背离 |
| 核心矛盾清晰度 | 单一明确的主线 | 有主线但有干扰因素 | 多个矛盾并存 |
| 数据质量 | 数据新鲜、完整 | 部分数据滞后 | 关键数据缺失 |
| 市场定价 | 未充分定价 | 部分定价 | 充分定价或过度定价 |

#### 6.7 AI 生成报告的强制约束

**⚠️ 以下规则必须严格遵守：**

1. **TL;DR 规范 (Full模式必须)**:
   - 不超过 30 个字
   - 必须包含：当前体制判断 + 核心动作方向 + 关键防范点
   - 示例: "滞胀担忧升温，规避成长股，利用低VIX买SPY看跌期权防范非农超预期"

2. **置信度必须标注 (Full模式)**:
   - 在 TL;DR 后和交易传导模块标注置信度
   - 必须给出一句话理由
   - 🔴低置信度时必须提示"多看少动"

3. **模式严格区分**:
   - Brief 模式：精简、快速消费、不超过 500 字
   - Full 模式：详尽、深度分析、包含所有模块

4. **标签必须使用**:
   - Full 模式必须在标题行标注宏观体制
   - 核心矛盾必须标注生命周期阶段（如适用）
   - 背离/Pain Trade/系统性风险必须使用对应预警标签

5. **数据表格规范**:
   - 所有数值必须对齐小数点
   - 涨跌必须用 +/- 明确标注
   - 缺失数据标注 `[N/A]` 或 `[数据暂不可用]`

6. **禁止空洞表述**:
   - ❌ "市场波动加大" → ✅ "VIX从15升至22，涨幅47%"
   - ❌ "关注油价走势" → ✅ "WTI $85支撑，跌破则下看$80"
   - ❌ "可能有风险" → ✅ "若CPI>3.5%，美债空头将被迫回补"

7. **因果链必须完整**:
   - 每个重要论断必须有"发生了什么 → 为什么重要 → 影响什么"

8. **时效性标注**:
   - 若数据滞后超过1个交易日，必须标注 `[数据滞后 T-N]`
   - 若有今日重大数据待公布，必须在关注清单中提及

9. **语言规范**:
   - 默认中文输出
   - 技术术语可混用英文 (Fed, DXY, CPI, FOMC, etc.)
   - 若用户用英文提问，则全英文输出

10. **质量检查隐式执行**:
    - AI 在生成过程中隐式运行 6.6 检查清单
    - 不在最终报告中输出自检过程
    - 只要结果符合检查项即可

11. **免责声明必须包含**:
    - 每份报告结尾必须有免责声明
    - 不得给出具体买卖建议，只提供分析框架
