# /daily-macro Command

Generate a daily macro analysis report covering Fed policy, commodities, treasury rates, FX, central banks, and breaking news.

## Usage

```
/daily-macro [--brief|--full]
```

## Options

- `--brief` (default): Quick summary with key data points
- `--full`: Comprehensive report with tables, trade implications, and historical playbook

## Examples

```
/daily-macro
/daily-macro --brief
/daily-macro --full
```

## Natural Language Triggers

This skill also activates on:
- 每日宏观 / daily macro / 今日市场 / 宏观分析 / macro briefing
- 市场速览 / market overview / 宏观速报

## Output

Markdown report covering:
1. Fed Dynamics (利率、官员表态、预期)
2. Commodities (原油、黄金、铜)
3. Treasury Rates (收益率曲线、美日利差)
4. FX Markets (DXY、主要货币对)
5. Central Bank Watch (全球央行动态)
6. Breaking News (地缘政治、突发事件)

### Enhanced Features (--full mode)
- Expectation Gap Alerts (预期差预警)
- Cross-Asset Divergence Detection (跨资产背离观察)
- Trade Implication Framework (交易传导推演)
- Historical Playbook (历史剧本复盘)
